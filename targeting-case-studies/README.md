# Targeting Case Studies

This section showcases complex customer segmentation and targeting strategies implemented for enterprise email campaigns.

### 3. [Product Activation Recurring Campaigns](./product-activation-targeting.md)

## Key Methodologies

- **Multi-Cohort Segmentation**: Advanced SQL techniques for complex customer categorization
- **Partner Suppression Logic**: Sophisticated exclusion rules for compliance
- **Dynamic Date Calculations**: T-minus logic for time-sensitive targeting  
- **Feature-Based Targeting**: Product capability detection for personalized messaging
- **Regulatory Compliance**: Multi-market legal requirement implementation
